# Newsletter Builder V0.1.2 使用手册

> 浏览器打开即用的学术期刊简报制作工具

---

## 快速上手

### 第一步：打开工具

1. 下载 `index.html` 文件
2. 双击用浏览器打开

> 不需要安装任何软件，不需要注册账号，不需要联网。

### 第二步：选择起点

打开后你会看到一个欢迎弹窗：

| 选项 | 做什么 |
|------|--------|
| **加载示例模板** | 直接加载一个做好的期刊模板，看看效果再改 |
| **新建空白画布** | 从零开始，自己搭内容 |
| **本次关闭** | 关掉弹窗，继续上次的编辑 |
| **今日关闭** | 今天不再显示这个弹窗 |

> 如果关掉了欢迎弹窗但想重新打开，刷新页面即可。

### 第三步：开始编辑

- 选了**示例模板** → 自动加载一个期刊简报，直接改文字和图片就行
- 选了**空白画布** → 去左侧面板的"组件"里点按钮，往画布上添加内容

---

## 怎么切换模式

顶部的三个按钮控制工作模式，随时可以切换，内容不会丢：

| 模式 | 按钮 | 用在哪 |
|------|------|--------|
| **代码模式** | 点"代码" | 你会写 HTML，想手写代码 |
| **设计模式** | 点"设计" | 你不会写代码，用鼠标拖拽排版 |
| **预览模式** | 点"预览" | 做完了看看最终效果 |

## 设计模式（最常用）

### 界面长这样

```
┌──────────┬──────────────────────┬──────────────┐
│ 左侧面板  │     中间画布          │  右侧属性面板  │
│          │                      │              │
│ [组件]    │  你添加的内容在这里    │  [上移] [下移] │
│ [模板]    │  可以拖拽调整顺序      │  [复制] [删除] │
│ [样式]    │                      │              │
│ [检查]    │                      │  文字: [____] │
│          │                      │  颜色: [____] │
└──────────┴──────────────────────┴──────────────┘
```

### 怎么添加内容

1. 点左侧的 **"组件"** 标签
2. 看到一排方块，每个代表一种内容类型（标题、文字、图片、按钮等）
3. **点一下想要的方块** → 内容自动添加到画布上

### 19 种内容块

| 名称 | 适合放什么 |
|------|-----------|
| **标题** | 文章标题、栏目名称 |
| **文本** | 正文段落 |
| **图片** | 单张图片（封面、插图） |
| **图文混排** | 图片和文字并排 |
| **按钮** | 引导点击（如"投稿"、"了解更多"） |
| **分隔线** | 内容分区 |
| **空白间距** | 拉开上下内容的距离 |
| **列表** | 条目列举 |
| **引用** | 名人名言、摘要 |
| **提示框** | 重要通知、注意事项 |
| **图片集** | 多张图片展示 |
| **作者信息** | 作者名、单位、头像 |
| **期刊封面** | 展示期刊封面图片 |
| **目录** | 文章目录 |
| **论文列表** | 多篇论文条目 |
| **论文卡片** | 单篇论文展示（标题、作者、摘要、链接） |
| **事件卡片** | 会议、征稿截止日期 |
| **横幅** | 促销、通知横幅 |
| **页脚** | 版权信息、社交链接 |

### 怎么修改内容

1. **点击画布上的内容块** → 右侧出现属性面板
2. 在属性面板里改文字、换图片、调颜色
3. 改完立即生效，不用保存

### 怎么调整顺序

**方法一（拖拽）**：
1. 鼠标移到内容块上，左侧出现 `⋮⋮` 图标
2. 按住 `⋮⋮` 上下拖动
3. 看到蓝色线出现时松开，内容就放到那里

**方法二（按钮）**：
1. 选中内容块
2. 右侧属性面板点 [上移] 或 [下移]

### 怎么复制/删除

- **复制**：选中内容块 → 点属性面板的 [复制] 按钮
- **删除**：选中内容块 → 点属性面板的 [删除] 按钮

### 搞错了怎么办

- 点顶部的 **撤销** 按钮（←箭头），可以回退到上一步
- 点 **重做** 按钮（→箭头），可以恢复撤销的操作

## 怎么调样式

### 调颜色

1. 点左侧的 **"样式"** 标签
2. 在"颜色"区域操作：

| 你想做什么 | 怎么做 |
|-----------|--------|
| 换主题色 | 点颜色框选颜色，或直接输入色号（如 `#667eea`） |
| 一键换配色方案 | 点下面的色块（科技蓝、电商红、教育绿...共 12 套） |
| 自动生成配色 | 点 [类比色] [互补色] [三角色]，系统自动算 |

### 调字体

1. 点左侧的 **"样式"** 标签
2. 在"字体"区域操作：

| 你想做什么 | 怎么做 |
|-----------|--------|
| 换字体风格 | 下拉选择（系统、苹方、微软雅黑、衬线、等宽） |
| 调标题大小 | 拖动 H1/H2/H3 的滑块 |
| 调正文字体 | 拖动字号、行高的滑块 |

### 调间距

1. 点左侧的 **"样式"** 标签
2. 在"间距"区域拖动滑块：

| 滑块 | 调什么 |
|------|--------|
| 容器宽度 | 内容区域的宽窄（480~680px） |
| 区块间距 | 内容块之间的上下距离 |
| 内边距 | 内容离边框的距离 |

## 导出和发布

### 下载 HTML 文件

1. 点顶部工具栏的 **"导出"** 按钮
2. 浏览器自动下载一个 `newsletter.html` 文件

### 怎么发布

**嵌入期刊网站**：
1. 用记事本打开下载的 `newsletter.html`
2. 全选复制全部内容
3. 登录期刊网站后台，找到 HTML 编辑器，粘贴进去

**用 GitHub Pages 发布**：
1. 把 `newsletter.html` 上传到 GitHub 仓库
2. 启用 GitHub Pages
3. 访问 `https://你的用户名.github.io/仓库名/newsletter.html`

**嵌入到已有网页**：
```html
<iframe src="newsletter.html" width="100%" height="800" style="border:none;"></iframe>
```

## 检查质量

### 检查暗色模式效果

1. 点左侧的 **"检查"** 标签
2. 勾选 **"邮件暗色客户端预览"**
3. 画布切换为深色背景，看看内容是否清晰
4. 取消勾选恢复
5. 导出时会自动带上暗色适配

### 检查文字对比度（无障碍）

1. 点左侧的 **"检查"** 标签
2. 点 **"WCAG-AA 审计"** 按钮
3. 下方会出现审计报告：
   - 红色标题：**必须修复** — 文字和背景对比度不够
   - 橙色标题：**建议修复** — 对比度可以更好
4. 每条问题会告诉你：哪个组件有问题、当前对比度、建议改成什么颜色
5. 按建议去属性面板改颜色，改完再点一次审计复查

### 发送前检查

1. 点左侧的 **"检查"** 标签
2. 点 **"运行检查"** 按钮
3. 系统检查 7 项内容，每项显示通过或失败：
   - 图片是否有效
   - 链接是否有效
   - 图片是否有替代文字
   - 手机端能否正常显示
   - 颜色对比度是否合规
   - 邮件客户端是否兼容
   - 垃圾邮件评分
4. 根据失败项调整内容

## 版本历史（防止改错）

系统会自动记录每次编辑，存到最多 50 个版本。

### 查看历史版本

1. 点左侧的 **"检查"** 标签
2. 往下翻到底，看到 **"版本历史"**
3. 列表显示每次编辑的时间和内容数量

### 恢复到某个版本

- 直接点你想恢复的那一行
- 画布立即回到那个版本

### 删除历史版本

- 鼠标移到某一行，右侧出现 × 按钮 → 点它删除这一条
- 点标题旁边的 🗑 图标 → 全部清空

## 常见问题

### 我做的东西存哪里了？

全部存在你的浏览器里：
- 设计内容、模板 → 浏览器本地存储（不清缓存就不丢）
- 版本历史 → 浏览器 IndexedDB（最多 50 个版本，旧的自动淘汰）

**重要**：这些数据只在你当前用的这台电脑的这个浏览器里。换个电脑、换个浏览器就没了。建议定期点"导出"备份。

### 能换电脑接着做吗？

不能。数据在浏览器本地，不跨设备同步。你需要在一台电脑上做完，导出 HTML 带走。

### 版本历史找不到刚才的版本？

- 只有修改内容才会自动记录，纯拖拽不会记录
- 超过 50 个版本后，最旧的会被自动删除
- 浏览器存储空间不够也可能保存失败

### 审计报告里一堆 "⋮" 的失败项？

那是编辑器的拖拽手柄图标，不影响最终导出的效果。只看带文字的内容块就行。

### 想清空所有数据重新开始？

浏览器按 F12 → 选 Application（应用） → 清除 localStorage 和 IndexedDB。
